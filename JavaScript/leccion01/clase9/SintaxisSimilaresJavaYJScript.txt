Operador de aumento
++
Ejemplo: Let z = ++a; // PreIncremento (se le asigna a z el valor de a incrementado por 1)
Ejemplo2: z = a++; // PostIncremento (si a = 2, z se queda con el 2 pero a termina siendo 3)

Opeador de decremento
--
Ejemplo: Let z = --a; // PreDecremento
Ejemplo2: z = a--; // PostDecremento
